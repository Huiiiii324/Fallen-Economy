# Auction House

`/ah` is built into Fallen Economy and uses `$`.

Commands:

```text
/ah
/ah sell <price>
/ah cancel <id>
/ah sort <newest|oldest|price_asc|price_desc|amount>
```

Data file:

```text
plugins/FallenEconomy/auctions.yml
```

When purchase confirmation is enabled, buyers must confirm before `$` is withdrawn.
